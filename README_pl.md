## Przegląd

Font Compare is an easy way to dynamically compare locally hosted fonts. Everything is run client-side; no user data stored on the server. 


**Dostarczona wersja:** 1.4~ynh1

**Demo:** <https://tenacitydc.com/fontcompare>
